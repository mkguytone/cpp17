定数式（constant expression）は C++14 では const に近い意味合いが強く実際に「式」という面では、条件式を定数式として評価できない弱みがありました。

C++17 以降は条件式にも使えるようになりました。

if constexpr (条件式) {
  // 処理部
}

この条件文は使い勝手はあまりよくはないです。

なぜなら条件式の中身は定数式でないと駄目だからです。
